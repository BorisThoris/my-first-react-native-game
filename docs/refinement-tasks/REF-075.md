# REF-075: `CodexScreen` search debounce and highlight perf
> **Rollup:** Addressed in codebase per [COMPLETION.md](./COMPLETION.md) (evidence row **071–080**).


## Problem

Live search over large encyclopedia text can jank input if unthrottled.

## Proposed work

- Debounce 100–150ms; Web Worker for fuzzy match if needed (optional).
- Cancel in-flight search when query changes.
