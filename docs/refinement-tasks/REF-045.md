# REF-045: `useDragScroll` — scroll chaining and modal wheel capture
> **Rollup:** Addressed in codebase per [COMPLETION.md](./COMPLETION.md) (evidence row **041–050**).


## Problem

`src/renderer/hooks/useDragScroll.ts` enables drag-to-scroll regions. Nested scrollables and open modals can steal wheel events or cause double-scroll.

## Proposed work

- Document intended behavior when `OverlayModal` is open; add manual test checklist or RTL test for wheel propagation.
- Respect `overscroll-behavior` / reduced motion where applicable.
