## Summary

**Tile focus assist** (`settings.tileFocusAssist`) dims **hidden** tiles that are neither the single open flip nor an **orthogonal** grid neighbor of that tile. The dim set is computed only during **`run.status === 'playing'`** with **exactly one** id in `board.flippedTileIds`, and only when the open tile’s `state` is **`flipped`**. If there is nothing to dim (e.g. a **1×2** board with one flip — only the neighbor remains), the helper returns **`undefined`** so upstream `useMemo` does not hold an empty `Set`.

## Data flow

| Step | Responsibility |
|------|----------------|
| `computeFocusDimmedTileIds` | `ReadonlySet<string>` of tile ids to dim, or **`undefined`** (assist off, wrong phase, or no dimmable tiles). |
| `GameScreen` | `useMemo` from `run.board`, `run.status`, `settingsTileFocusAssist` → `dimmedTileIds={focusDimmedTileIds}`. |
| `TileBoard` | Forwards **`dimmedTileIds`** to `TileBoardScene` only (no separate DOM tile layer; canvas is `aria-hidden`; assist is visual). |
| `TileBoardScene` | `tileBezelRows`: `focusDimmed: Boolean(dimmedTileIds?.has(tile.id))` per tile. |
| `TileBezel` / `advanceTileBezelFrame` | `dimTarget = focusDimmed && !faceUp && tile.state === 'hidden'` → damped brightness/opacity on card materials. Peek / preview **faceUp** overrides prevent dimming faces that are shown for reading. |
