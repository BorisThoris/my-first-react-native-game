# REF-071: CSS modules vs `theme.ts` — injection order and specificity
> **Rollup:** Addressed in codebase per [COMPLETION.md](./COMPLETION.md) (evidence row **071–080**).


## Problem

## Proposed work

- Inventory literals still in CSS modules; migrate to `var(--token)` from a single `:root` theme layer.
- Story or visual test for Meta/Game/Settings in dark theme.
