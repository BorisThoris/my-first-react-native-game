# Symbol set guidelines

- **One concept per tile** — symbol + label should reinforce the same idea (no unrelated abbreviations on the same face).
- **Contrast** — pair symbols must be distinguishable at a glance on the smallest tile size; avoid low-contrast hue pairs (red/orange adjacent).
- **Cognitive load** — prefer closed shapes and short labels (≤6 chars) for numeric sets; avoid confusable glyphs (O/0, I/1/l).
- **Accessibility** — do not rely on color alone; shape and label must carry meaning for color-blind players.
- **Skins / sets** — new themes must pass the checklist in [PICTURE_SUPERIORITY_CHECKLIST.md](./PICTURE_SUPERIORITY_CHECKLIST.md).
