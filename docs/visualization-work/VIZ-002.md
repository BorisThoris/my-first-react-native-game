# VIZ-002: Dev visualization gallery (`pairKey` × `OverlayDrawTier`)

## Problem

E2E hashes and local PNG bakes make regressions **detectable** but not easy to **eyeball**. A single place to compare tiers and pair keys speeds art and shader reviews.

## Proposed work

- **Shipped:** DEV sandbox [`ProceduralIllustrationGallerySandbox.tsx`](../../src/renderer/dev/ProceduralIllustrationGallerySandbox.tsx) — `?devSandbox=1&fx=proceduralGallery` (see [README](./README.md)). Pair keys mirrored in [`illustrationRegressionPairKeys.ts`](../../src/renderer/dev/illustrationRegressionPairKeys.ts) with a Vitest guard vs the E2E fixture.
- **Remaining / optional:** keyboard shortcuts for tier buttons, pairKey filter field, side-by-side tier compare for one key, Playwright smoke (`data-e2e-*`). Track in [EXPLICIT_50_AGENTS.md](./EXPLICIT_50_AGENTS.md) agents 11–20.
