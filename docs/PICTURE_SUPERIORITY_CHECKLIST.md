# Picture superiority checklist (art / UI)

Use when adding themes, larger tiles, or picture-based pairs.

- [ ] Symbol readable at **minimum** tile size in gameplay (mobile camera + fallback grid).
- [ ] `label` legible when shown (wide recall / future modes).
- [ ] Silhouette / silhouette_twist mutator: silhouette still suggests the correct category.
- [ ] Atomic pair variants: adjacent pairs on the board are visually separable (border hue or shape ring).
- [ ] Motion: no critical information only in animation; static frame must be identifiable.
