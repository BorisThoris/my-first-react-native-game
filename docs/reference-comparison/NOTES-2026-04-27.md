## P0 visual issues (in scope for CSS / chrome)

- **Main menu:** Scene + procedural layers compete with the title and CTA stack; meta card band reads busy relative to a single focal column; any residual grid/noise over the hero should stay sub-threshold.
- **Choose Your Path:** Stacked scrims and dual scene layers read muddy; mode cards need clearer panel separation and calmer background so typography pops.
- **Gameplay:** Stage backdrop (vignette + torch ring) can overpower the board; HUD segments benefit from clearer token-driven contrast and less redundant warm `rgba` stacks.
